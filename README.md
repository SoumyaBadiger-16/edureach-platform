# 🎓 EduReach AI

**EduReach AI** is an AI-powered educational platform designed to provide personalized learning experiences for students. It combines artificial intelligence, modern web technologies, and an intuitive interface to help students learn more efficiently while assisting educators with academic management.

## 🚀 Features

* 🤖 AI-powered personalized learning recommendations
* 📚 Smart study path generation based on student goals
* 💬 AI chatbot for instant academic assistance
* 📝 Question paper analysis and preparation guidance
* 📊 Student progress tracking and performance analytics
* 🎯 Placement preparation assistant
* 📅 Academic event and deadline reminders
* 🔍 Intelligent search for learning resources
* 🌙 Responsive and user-friendly interface

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/EduReach-AI.git
```

Navigate to the project directory:

```bash
cd EduReach-AI
```

Install dependencies:

```bash
npm install
```

Start the frontend:

```bash
npm run dev
```

Start the backend:

```bash
uvicorn main:app --reload
